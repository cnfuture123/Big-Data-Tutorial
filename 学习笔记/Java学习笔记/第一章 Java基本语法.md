# Java基本语法

## 标识符

  - 定义：Java对各种变量、方法和类等命名时使用的字符序列称为标识符。
  - 规则：
    - 由26个英文字母，0-9，_或$组成
    - 数字不可以开头
    - 严格区分大小写，长度无限制
    - 不能包含空格
  - 命名规范：
    - 包名：多单词组成时所有字母都小写
    - 类名、接口名：单词首字母大写
    - 变量名、方法名：第一个单词首字母小写，第二个单词开始每个单词首字母大写
    - 常量名：所有字母都大写。多单词时用_连接
    
## 变量

  - 定义：
    - 内存中的一个存储区域，用于保存数据
    - 变量是程序中最基本的存储单元，包含：变量类型、变量名和存储的值
  - 注意点：
    - Java中变量必须先声明，后使用
    - 使用变量名访问这块区域的数据
    - 变量的作用域：其定义所在的一对{}内
  - 声明和赋值变量：
    - 语法：<数据类型> <变量名> = <初始值>
    - 例如：int num = 10;
