# 大数据概述

  - 大数据主要解决海量数据的存储和分析计算问题。
  - 按顺序给出数据存储单位：Bit -> Byte -> KB -> MB -> GB -> TB -> PB -> EB -> ZB -> YB -> BB -> NB -> DB。
  
## 大数据特点

  - Volume(大量)
  - Velocity(高速)
  - Variety(多样)：结构化数据，非结构化数据。
  - Value(低价值密度)：价值密度的高低与数据总量的大小成反比。
  
## 大数据应用场景

  - 物流仓储
  - 零售
  - 旅游
  - 商品广告推荐
  - 保险
  - 金融
  - 房产
  - 人工智能
