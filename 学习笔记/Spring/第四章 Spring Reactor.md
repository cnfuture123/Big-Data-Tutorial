# Spring Reactor

## 简介

  - 概述：
    - Project Reactor和Spring技术栈组合可以共同构建企业级，低延时，高吞吐量的响应式系统
    - 响应式处理是一种范式，使开发者可以构建非阻塞、异步的应用，并且支持反压机制
  - Project Reactor：
    - Project Reactor是一个Spring生态中响应式技术栈的基础，用于Spring WebFlux, Spring Data, and Spring Cloud Gateway等
